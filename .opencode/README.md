# OpenCode Deep Agent Profile

This folder follows the `.opencode` workspace layout used by OpenCode, but only the AI ML onboarding Deep Agent profile is enabled.

Included:

- `agent/deep-agent.md`: Deep Agent instructions for AI/ML onboarding, MLflow validation, Job Template planning, and approval-gated fixes.
- `opencode.jsonc`: Workspace references for vendored DeepAgents libs and local onboarding skills.
- `tui.json`: Minimal TUI config with no upstream plugins enabled.
- `themes/deep-agent.json`: Dark terminal theme aligned with the POC console UI.

Not included:

- Upstream GitHub triage or duplicate PR agents.
- Effect-specific skills.
- Upstream plugins or GitHub tools.
